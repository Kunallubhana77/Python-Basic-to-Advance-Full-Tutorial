## ğŸš€ Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kunallubhana77/Python-Basic-to-Advance-Full-Tutorial.git
   cd Python-Basic-to-Advance-Full-Tutorial
   ```

2. **Install Python** (3.8 or higher)
   ```bash
   python --version  # Check if Python is installed
   ```

3. **Install required libraries** (for data science sections)
   ```bash
   pip install numpy pandas matplotlib
   ```

```bash

python Practice.py/Basic\ \&\ Logic/first.py
```

### Quick Peek at What You'll Learn

**Day 1: Hello World**

**Day 3: Pattern Printing**
```python
for i in range(1, 6):
    print("* " * i)
# Output:
# *
# * *
# * * *
# * * * *
# * * * * *
```

**Day 8: Dictionary Operations**
```python
student = {"name": "Alice", "marks": 85}
print(f"{student['name']} scored {student['marks']}")
```

**Day 11: Custom Exceptions**
```python
class InsufficientFundsError(Exception):
    pass

def withdraw(balance, amount):
    if amount > balance:
        raise InsufficientFundsError("Not enough funds!")
    return balance - amount
```

**NumPy: Array Operations**
```python
import numpy as np
arr = np.array([1, 2, 3, 4, 5])
print(f"Mean: {arr.mean()}, Sum: {arr.sum()}")
```

**Pandas: Data Analysis**
```python
import pandas as pd
df = pd.DataFrame({'Name': ['Alice', 'Bob'], 'Marks': [85, 92]})
print(df[df['Marks'] > 80])
```
